#Interactive Resume

######Project Overview

Project #2 of the [Front-End Web Developer Nanodegree](https://www.udacity.com/course/front-end-web-developer-nanodegree--nd001).

The task was to develop an interactive resume application that reads your resume content from a JSON file and dynamically displays that content within a provided template. Requirements were to use objects, functions, conditionals, and control structures to compose the content that will display on the resume.
